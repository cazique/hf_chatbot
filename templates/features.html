Características Principales

ImmoDoc OCR ofrece un conjunto de herramientas para la gestión de documentos inmobiliarios. A continuación se presentan las funcionalidades más destacadas del sistema:

  • OCR y Reconocimiento de Texto

    Digitaliza documentos escaneados y obtén su contenido en formato editable mediante Tesseract OCR integrado.

  • Optimización de PDFs

    Reduce el tamaño de tus PDFs manteniendo la calidad, ideal para almacenamiento y envío eficiente.

  • Panel de Control

    Visualiza estadísticas de uso, documentos procesados y espacio ahorrado en una interfaz intuitiva.

  • Integración con IA

    Configura proveedores de modelos de lenguaje (LLM) como Ollama y otros para realizar consultas inteligentes sobre tus documentos.

  • Carga y Procesamiento por Lotes

    Sube uno o varios archivos a la vez con soporte de arrastrar y soltar; el sistema procesará cada documento y optimizará el tamaño si se solicita.

  • Extracción de Texto

    Obtén el texto existente de un PDF digital sin aplicar OCR, ideal para documentos que ya contienen texto embebido.

  • Conversión a Imágenes

    Transforma las páginas de un PDF en imágenes individuales para inspección rápida o creación de miniaturas.

  • Gestión de Usuarios

    Crea y administra usuarios con diferentes temas y esquemas de color; configura permisos de administrador para acceder a opciones avanzadas.

  • Historial y Logs

    Consulta un listado de documentos procesados, tamaños originales y optimizados, fechas y usuarios que realizaron la acción.
